# 🎮 Stone Paper Scissors

A terminal-based Stone Paper Scissors game built in Python.

## 🕹️ How to Play
- Enter stone, paper or scissors
- Computer randomly picks its choice
- Winner is decided based on classic rules
- Type 'quit' to exit and see final score

## 📊 Features
- Score tracker (wins, losses, draws)
- Input validation
- Replay without restarting

## 🛠️ Built With
- Python
- Random module
